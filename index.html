n8n_chat_app reads Instagram messages and may access message content, metadata, and basic Instagram profile information required for API operations. No other Instagram data is accessed.

Data is used exclusively to retrieve and process Instagram messages as part of your n8n automations. It is not used for advertising, analytics, or any secondary purpose.

5. Data Sharing

We do not sell or share data with third parties. Data is only exchanged between your n8n workflows and the Instagram API.

6. Data Security

All communication with Instagram APIs is done over HTTPS. Users are responsible for securing their own n8n environment.
